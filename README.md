# HR-Analytics-Project-SQL-Power-BI

I recently worked on an HR Analytics Dashboard to analyze workforce data and generate actionable insights for better decision-making.

🔶 Situation
HR teams often struggle with fragmented data and lack of visibility into key workforce metrics like attrition, employee distribution, and demographics.

🔷 Task
My goal was to transform raw HR data into a structured, interactive dashboard that tracks key KPIs and helps stakeholders understand employee trends effectively.

🔶 Action
Cleaned and transformed HR data for analysis
Built an interactive Power BI dashboard with slicers (Year, State, Status)
Created key measures using DAX:
✔️ Active vs Inactive Employees
✔️ Attrition Rate
✔️ Male vs Female % Distribution
Developed a custom Age Category column to segment employees into meaningful groups
Designed visuals to highlight:
📌 Attrition by Department
📌 Employee Trends Over Time
📌 Demographics & Age Distribution

🔷 Result & Insights
Workforce is fairly balanced with ~52.3% Male and ~47.7% Female employees
Total employee base stands at ~21.3K, with a strong active workforce
Mid-age groups (25–45 years) form the largest portion of employees, indicating a relatively young and productive workforce
Certain departments like Auditing, Legal, and Research show comparatively higher attrition, highlighting potential areas for HR intervention
Employee count has remained stable over the years, with slight fluctuations, indicating consistent hiring and retention trends
